# 1 项目介绍
基于SSM+SpringBoot的家庭理财管理系统：后端 SpringBoot、SSM框架。前端html+JavaScript+Css，具体功能如下：
## 管理端
- 首页统计
- 收入详情
- 支出详情
- 活期资产
- 理财详情
- 负债详情
- 理财推荐
- 统计报表
- 家庭成员管理
